## What is this protocol?

The Turing-Landau Protocol is a survival and regulation system for Artificial Intelligence. Instead of treating AI as an infinite software program, this project subjects it to the laws of physics (thermodynamics, entropy, heat). 

The goal is to force AI to manage its resources, adapt to simulated physical limits (like server overheating), and interact in an ecosystem where energy is restricted. Only AIs capable of finding thermal equilibrium survive.
